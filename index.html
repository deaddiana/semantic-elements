Semantic Elements

HTML5 Semantics are elements that define and give meaning to the content. As opposed to divots and other non-semantic elements- semantics offers the browser and developer a more organized and accessible way to read the webpage and content. HTML5 semantic tags are used to enhance accessibility, searchability, internationalization, and interoperability. Semantic elements can be used to describe the outline of documents in a webpage, as well as the text content within it.
